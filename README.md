# LCD Display Examples (Lab 4)

This folder contains Arduino sketches and notes for Lab 4: LCD & OLED display.

Files:

- `Diagram.json` - wiring diagrams for the LCD example.
- `Libraries.txt` - libraries required for the LCD sketches.
- `Sketch.ino` - main example sketch for the LCD display.

Usage:

1. Install the libraries listed in `Libraries.txt` (for Arduino IDE or your package manager).
2. Open `Sketch.ino` in the Arduino IDE or PlatformIO and upload to your board.
